# Organización de contenido

- Mantén el mismo slug que la versión predeterminada (`content/about` → `content/es/about`).
- Reutiliza frontmatter existente para conservar referencias de búsqueda.
- Actualiza enlaces internos para apuntar a las rutas localizadas.

Estas páginas de ejemplo no reemplazan la lógica del generador; sirven como referencia cuando definas tu propia estrategia de internacionalización.
