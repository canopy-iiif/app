<Container>
  <h1>Notes & credits</h1>
  <p>
    Treat this page as a lightweight colophon. Capture the production context,
    cite additional readings, and outline how the homepage should evolve. Because
    it is regular markdown, you can stack sections, include lists, or add
    callouts without touching React code.
  </p>
  <h2>Production notes</h2>
  <ul>
    <li>
      Replace the Newsreader font in `_app.mdx` if you need a different mood.
    </li>
    <li>
      Update `canopy.yml` with your collections before publishing; this template
      uses the same fixtures as the generic starter to stay predictable.
    </li>
    <li>
      The homepage derives its citations from `referencedManifests`. Keep that
      list in sync so `<ReferencedItems />` and `<References />` stay accurate.
    </li>
  </ul>
  <h2>Where to expand</h2>
  <ol>
    <li>Add new markdown files under `content/` when you need additional essays.</li>
    <li>Create `_layout.mdx` files per section to introduce navigation sidebars.</li>
    <li>
      Layer footnotes by writing markdown references or embedding a small React
      component if you need interactivity.
    </li>
  </ol>
  <h2>Colophon</h2>
  <p>
    Designed with Canopy IIIF. Fonts are loaded via Google Fonts, but you can
    bundle your own type by importing CSS inside `_app.mdx` or `app/styles/`.
    The default robots file (in `assets/robots.txt`) blocks indexing so you can
    work in private until you are ready to go live.
  </p>
</Container>
