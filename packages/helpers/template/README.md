# Canopy Starter Template

### This template is ready for your new Canopy IIIF project!

Follow the getting started guide at https://canopy-iiif.github.io/app/docs/guides/getting-started to learn how you can edit `canopy.yml` to point to your IIIF collection, create pages in the `content/` directory, and deploy your project to the internet.

## Docs

For full documentation, examples, and advanced configuration, see the main project at https://canopy-iiif.github.io/app/docs.

## Updates

Canopy IIIF is designed to be updated easily through an external package. Get the latest features from the core project, along with bug fixes and updates:

```sh
# Update Canopy IIIF
npm install @canopy-iiif/app@latest
```

## Contributing

If you are interested in helping shape the project, issues and pull requests are always welcome. Project contributions should be made to the main repository at https://github.com/canopy-iiif/app. The core Canopy IIIF package is defined in the `/packages/app` directory.
