# canopy-iiif.github.io

This repository hosts the build output for the public Canopy IIIF documentation site. The contents are managed automatically by https://github.com/canopy-iiif/app whenever a new release is published.

- `/index.html` is rendered from `packages/helpers/org/root/index.mdx` in the main repo.
- `/app/` is the compiled documentation + demos generated from `site/`.

- Latest docs: https://canopy-iiif.github.io/app/
- Source code: https://github.com/canopy-iiif/app
- Template for new projects: https://github.com/canopy-iiif/template
